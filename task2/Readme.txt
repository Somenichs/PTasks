Запуск - python task2.py path_to_file1 path_to_file2

По условию задания 
Формат файла1:
1 1
5
Где {1 1} - координаты окружности, {5} - радиус окружности

Формат файла2:
0 0
1 6
6 6
Где каждая строка содержит 2 координаты x и y

Вывод результатов в консоли в формате:
1
0
2
Где 
0 - точка лежит на окружности
1 - точка внутри
2 - точка снаружи.
