Запуск - python task3.py path_to_file1 path_to_file2 path_to_file3
Где path_to_file1 - путь до файла, который содержит результаты прохождения тестов с уникальными id
path_to_file2 - путь до файла, который содержит структуру для построения отчета на основе прошедших тестов
path_to_file3 - путь до файла, в который записывается результат
